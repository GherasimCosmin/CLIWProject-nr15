Login

The user can login or, more simply, enter the site as a guest. This will prompt him to the next page, the menu page.

Menu

Here, we display a menu with the main functionalities of our website: "New", for drawing a completely new fractal, "Continue", for working on a previously saved fractal, "Download" to download your work as PNG or "Logout".

New

The "New" page provides the user with the main swiper, which contains a canvas (where the fractals will be displayed), and a some input lists to input data. The user can choose and change the number of iterations, the angle of the fractal, the axioms and rules by which it is build.

Clicking "Generate" will display the newly-built fractal in the canvas, while clicking on "Save" will save the fractal on the local storage.

Choosing a fractal from the "List" will display it in the "Generate" section, with already-completed input fields.
